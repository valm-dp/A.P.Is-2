#                      DNC WEATHER

### Projeto focado para exercicio das abilidades com APIs


Link para acesso a hospedagem no Netlify: https://c0nsum0-de-apis.netlify.app

## Tecnoligias e APIS utilizados

- VSCode 

- Figma 

- Opem.Meteo

- ViaCEp
